# AI-Powered Interview Coaching Application

## Project Overview
This application serves as an AI-powered interview coach designed to help users prepare for job interviews through interactive simulations, feedback, and personalized recommendations. Built with modern technologies, this application provides a seamless learning experience for users seeking to improve their interviewing skills.

## Features
- **Interactive Simulations**: Engage in mock interviews with AI-generated questions tailored to various industries and roles.
- **Real-Time Feedback**: Receive immediate feedback on your answers with suggestions for improvement.
- **Personalized Recommendations**: Get tailored resources and guidance based on your performance and areas of improvement.
- **Analytics Dashboard**: Track your progress and performance over time with easy-to-understand metrics.

## Tech Stack
- **Frontend**: React, TypeScript, Vite  
- **UI Components**: shadcn-ui  
- **Styles**: Tailwind CSS  

## Installation
To get started, clone the repository and install the dependencies:
1. Clone the repository:
   ```bash
   git clone https://github.com/Prem-91/interview-coach-ai-05.git
   ```
2. Navigate into the project directory:
   ```bash
   cd interview-coach-ai-05
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```

## Usage
To start the development server, run:
```bash
npm run dev
```
Open your browser and navigate to `http://localhost:3000` to see the application in action.

## Project Structure
- `/src` - Contains the source code for the application  
- `/components` - Reusable React components  
- `/pages` - Page components for routing  
- `/styles` - Application styles
